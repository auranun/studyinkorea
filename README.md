# Study in Korea App 🇰🇷📱

This is a React Native application designed to help Mongolian students who are planning to study in South Korea.  
It provides useful information like visa guides, cultural events, school tips, and more.

## Key Features
- Visa guide & document help
- Mongolian event calendar in Korea
- Part-time job tips
- Firebase integration
- Clean UI in React Native

## Installation
```bash
npm install
npx react-native run-ios # or run-android
